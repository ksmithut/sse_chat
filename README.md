# sse_chat

A simple chat server built in node to demonstrate how server-sent events works.

## Prerequisites

- [node](https://nodejs.org) v16
- [yarn](https://classic.yarnpkg.com/lang/en/) v1

## Installation

```sh
yarn
```

## Running

```sh
yarn start
```

For dev mode (auto restart on file change):

```sh
yarn start:dev
```

## Tests

```sh
yarn test
```
